# Я Знаю (iOS, Local Wi-Fi P2P)

Минимальный каркас игры по локальной сети (Host/Player) на SwiftUI + Network.framework.

## Что уже реализовано

- Splash-экран с авто-переходом.
- Выбор роли: ведущий или игрок.
- `NetworkManager` на `NWListener` / `NWConnection`:
  - запуск TCP сервера,
  - подключение по IP/порту,
  - JSON обмен сообщениями,
  - прием/отправка сообщений,
  - обработка потери соединения и авто-перезапуск/переподключение.
- Экран ведущего:
  - запуск сервера,
  - список игроков,
  - старт игры только от 2 игроков,
  - отправка вопроса игрокам,
  - прием ответов,
  - рассылка результатов.
- Экран игрока:
  - ввод ника, IP и порта,
  - ожидание вопроса,
  - `GameView` с 4 крупными вариантами ответа,
  - визуальная обратная связь (зеленый/красный).

## Структура проекта

- `quizIOS2/Models/GameModels.swift` — модели и типы сообщений.
- `quizIOS2/Network/NetworkManager.swift` — TCP/JSON networking.
- `quizIOS2/ViewModels/AppViewModel.swift` — состояние приложения и игровая логика.
- `quizIOS2/Views/Shared/*` — Splash и выбор роли.
- `quizIOS2/Views/Host/*` — экраны ведущего.
- `quizIOS2/Views/Player/*` — экраны игрока и `GameView`.
- `quizIOS2/ContentView.swift` — роутинг экранов по фазам.

## Быстрый запуск (на Mac в Xcode)

1. Открыть `quizIOS2.xcodeproj`.
2. Запустить приложение на двух устройствах/симуляторах в одной Wi-Fi сети.
3. На первом выбрать роль **Ведущий**, нажать **Запустить сервер**.
4. На втором выбрать **Игрок**, ввести IP ведущего и подключиться.
